# 虚拟环境

（1）安装依赖

```
pip install virtualenvwrapper-win
```

（2）创建虚拟环境

```
mkvirtualenv 虚拟环境名字
```

（3）激活虚拟环境

```
workon 虚拟环境名字
```

（4）退出虚拟环境

```
deactivate
```

（5）删除虚拟环境

```
rmvirtualenv 虚拟环境名字
```

（6）列出所有虚拟环境

```
lsvirtualenv
```

注：虚拟环境默认安装在用户下的Envs文件夹
